# Task 3: React Developer Tools Screenshots

## Required Screenshots

Save the following screenshots in this folder:

### 1. change_property.png

**What to show:**

- React DevTools Components tab open
- First NotificationItem component selected
- Props panel showing `type: "urgent"` (changed from "default")
- The notification appearing in RED color in the browser

### 2. profiler.png

**What to show:**

- React DevTools Profiler tab open
- Component render times displayed
- Flamegraph or Ranked view showing all components (App, Header, Footer, Login, Notifications, NotificationItem)
- Performance data clearly visible

## How to Complete This Task

See the detailed guide: [TASK_3_GUIDE.md](../TASK_3_GUIDE.md)

**Quick Steps:**

1. Install React Developer Tools browser extension
2. Run `npm start` in task_2, task_4, or task_5 dashboard folder
3. Open browser DevTools (F12)
4. Take screenshots as specified above
5. Save both screenshots in this directory
