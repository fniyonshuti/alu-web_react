# React Inline Styling

This project covers inline styling, CSS-in-JS with Aphrodite, responsive design, and animations in React applications.

## Learning Objectives

- The differences between using a CSS file and inline styling
- How to use a CSS-in-JS tool like Aphrodite
- How to use conditions within JS to apply different styles
- How to use responsive design and make the application show a different UI according to the screen size
- How to create small animations within the app

## Tasks

### 0. Inline styling

Modify the CourseListRow component to use inline styling for row background colors.

### 1. Install Aphrodite

Install Aphrodite and convert CSS files to Aphrodite inline styling.

### 2. Conditionally applying style

Use Aphrodite with conditions to apply different styles.

### 3. Responsive design

Make the application responsive using media queries.

### 4. Animation

Create animations for hover effects.
