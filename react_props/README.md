# React Props

This project focuses on learning React component properties, PropTypes, and component composition.

## Learning Objectives

- How to create basic React components using functions
- How to reuse components
- How to pass properties to components
- How to define types for components
- How to use Fragments
- When to use a key to improve a loop's performance

## Requirements

- All files interpreted/compiled on Ubuntu 18.04 LTS using node 12.x.x and npm 6.x.x
- Allowed editors: vi, vim, emacs, Visual Studio Code
- All files should end with a new line

## Tasks

### Task 0: Basic components

Split the monolithic App component into smaller reusable components (Header, Footer, Login).

### Task 1: Write tests for each component

Create comprehensive test suites for all components.

### Task 2: Split the Notifications component

Create NotificationItem component for better code reusability.

### Task 3: Checking the application using React extension

Use React Developer Tools to inspect and profile the application.

### Task 4: Props types & Default props & Shapes

Implement PropTypes validation and create CourseList components.

### Task 5: Shapes, Loops, and Keys

Create data shapes and implement dynamic rendering with arrays.
