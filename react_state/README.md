# React State

This project focuses on implementing state management in React applications, including local state, controlled components, context API, and React hooks.

## Tasks

- Implemented local state for `displayDrawer` in App component
- Created `handleDisplayDrawer` and `handleHideDrawer` functions
- Updated Notifications component to use these props
- Added tests for state changes

### Task 1: Controlled components and state callback

- Creating forms with controlled components in Login
- Implementing state callback for form validation
- Adding tests for form interactions

### Task 2: Context

- Creating React Context for user authentication
- Implementing login/logout with context
- Refactoring components to use context

### Task 3: Context consumer & advanced state

- Using Context Consumer in Footer
- Advanced state management for notifications
- Implementing markNotificationAsRead functionality

### Task 4: Introduction to React hooks
